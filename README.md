# Reach

Reach is an AI-powered mobile application that allows users to simulate conversations with people who are out of reach — such as past relationships, mentors, or loved ones — through personalized virtual personas.

## Tech Stack
- React Native (Expo)
- TypeScript
- FastAPI (planned)
- PostgreSQL / Supabase (planned)
- OpenAI API (planned)

## Features (Planned)
- Create personalized personas
- Memory-based conversation system
- AI-driven chat interface
- Emotional context modeling
